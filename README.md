# Futures App

Приложение для работы с фьючерсами: управление торгами и датами исполнения.

## Структура проекта

```
futures_app/
├── app.py                  # Точка входа приложения
├── db.py                   # Настройка базы данных
├── models.py               # Модели данных
├── services.py             # Бизнес-логика и сервисные функции
├── ui/                     # Модули пользовательского интерфейса
│   ├── dialogs/            # Диалоговые окна
│   ├── models/             # Модели таблиц для Qt
│   ├── pages/              # Страницы (вкладки) приложения
│   ├── styles/             # Стили и темы оформления
│   └── widgets/            # Пользовательские виджеты
└── data/                   # Директория с данными
    ├── dataisp.XLS         # Данные по датам исполнения
    └── F_usd.XLS           # Данные по торгам
```

## Модули UI

### Модели таблиц (`ui/models/`)
- `table_models.py` - модели для таблиц с торгами, датами исполнения и совмещенными данными

### Виджеты (`ui/widgets/`)
- `custom_widgets.py` - пользовательские виджеты (например, FuturesCodeComboBox)

### Диалоги (`ui/dialogs/`)
- `dialogs.py` - диалоговые окна для редактирования записей и импорта

### Страницы (`ui/pages/`)
- `trades_page.py` - страница для управления торгами
- `expirations_page.py` - страница для управления датами исполнения
- `combined_page.py` - страница с совмещенной таблицей

### Стили (`ui/styles/`)
- `theme.py` - настройки стилей и тем приложения

## Установка и запуск

1. Установите зависимости:
```
pip install -r requirements.txt
```

2. Запустите приложение:
```
python app.py
```
или
```
bash run.sh
```

## Функциональность

- Управление торговыми данными (добавление, редактирование, удаление)
- Управление датами исполнения фьючерсных контрактов
- Импорт данных из Excel файлов
- Просмотр данных в различных представлениях
