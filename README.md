# Futures Database Application

Приложение для управления базой данных фьючерсов с графическим интерфейсом.

## Возможности

- ✅ Управление торгами фьючерсов
- ✅ Управление датами исполнения
- ✅ Импорт данных из Excel файлов
- ✅ Автоматическое форматирование кодов фьючерсов (FUSD_XX_YY)
- ✅ Современный интерфейс с темной/светлой темой

## Установка и запуск

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd futures_app
```

### 2. Создание виртуального окружения
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Запуск приложения

**Способ 1: Через скрипт (рекомендуется)**
```bash
./run.sh
```

**Способ 2: Вручную**
```bash
source venv/bin/activate
python app.py
```

## Структура проекта

- `app.py` - основное приложение с GUI
- `models.py` - модели базы данных
- `db.py` - настройки подключения к БД
- `services.py` - бизнес-логика
- `futures.db` - база данных SQLite
- `requirements.txt` - зависимости Python
- `run.sh` - скрипт для запуска

## Использование

1. **Добавление торгов**: нажмите "Добавить" и введите данные
2. **Автозаполнение кода**: введите только цифры (например: `20 10` → `FUSD_20_10`)
3. **Импорт данных**: используйте кнопки импорта для загрузки из Excel
4. **Редактирование**: выберите строку и нажмите "Изменить"

## Технические детали

- **GUI Framework**: PySide6 (Qt6)
- **База данных**: SQLite
- **ORM**: SQLAlchemy 2.0
- **Python**: 3.9+
